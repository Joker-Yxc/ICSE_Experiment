# Documents

Project documents are organized by artifact type:

- `evaluation_icse2027.tex`
  - Main evaluation section source.
- `evaluation_icse2027_appendix.tex`
  - Evaluation appendix source.
- `README_experiment_setting.md`
  - Dataset protocol, settings, and result notes.
- `figures/`
  - Paper plots and editable framework-diagram sources.
- PDF and DOCX files
  - Reference manuscripts and revision materials retained for the experiment.

Generated paper exports are stored under the repository-level `output/`
directory.
